# 金属电子逸出功与质核比测定实验数据处理系统

## 简介
这是一个基于 Streamlit 开发的交互式物理实验数据处理网页应用，用于处理"金属电子逸出功与质核比测定"实验数据。

## 功能特点

### 🔬 实验内容
1. **电子逸出功的测定（理查森直线法）**
   - 不同温度下的 lgIa - √Ua 图线拟合
   - 理查森直线法 lg(I/T²) - 1/T 拟合
   - 逸出电势、逸出功计算与误差分析

2. **电子荷质比的测定（磁控法）**
   - 不同阳极电压下的 Ia - Is 曲线
   - 临界电流自动识别与手动调整
   - Un - Ic² 线性拟合
   - 电子荷质比计算与误差分析

### 🎨 UI 特点
- 简洁美观的蓝色调主题
- 分模块展示，条理清晰
- 响应式布局，适配不同屏幕
- 交互式图表，支持缩放、平移、悬停
- 实时计算，输入数据即更新结果

## 安装与运行

### 1. 安装依赖
```bash
pip install streamlit numpy pandas plotly scipy
```

### 2. 运行应用
```bash
streamlit run metal_electron_experiment.py
```

### 3. 访问应用
浏览器会自动打开，通常在 `http://localhost:8501`

## 使用说明

### 模块一：实验说明 📚
- 查看实验目的和原理
- 了解数据表格结构
- 查看内置的灯丝电流-温度对应关系表

### 模块二：逸出功数据输入 📊
- 点击"加载示例数据"一键填充演示数据
- 在表格中直接输入实验测量的 Ia 值（μA）
- 自动显示已输入数据统计

### 模块三：逸出功数据处理 📈
- **第一步：lgIa - √Ua 图线**
  - 显示7条不同温度下的拟合直线
  - 显示每条直线的拟合方程和 R² 值
  - 生成拟合结果表格

- **第二步：理查森直线**
  - 显示 lg(I/T²) - 1/T 拟合结果
  - 计算逸出电势 φ 和逸出功 W₀
  - 与理论值（4.54 eV）比较，计算百分误差

### 模块四：荷质比数据输入 ⚡
- 初始化数据表格（可自定义起始电流、间隔、行数）
- 动态添加/删除数据行
- 在表格中输入不同 Un 下的 Ia 值

### 模块五：荷质比数据处理 📉
- 绘制不同 Un 下的 Ia-Is 曲线
- 自动识别临界电流 Ic（可手动调整）
- Un - Ic² 线性拟合
- 计算电子荷质比 e/m
- 与理论值（1.76×10¹¹ C/kg）比较

### 模块六：结果汇总与导出 📋
- 两个实验的结果汇总
- 精度评价（好/一般/需检查）
- 所有数据支持 CSV 格式导出

## 内置参数

### 逸出功测定
- 灯丝电流 If：0.600, 0.625, 0.650, 0.675, 0.700, 0.725, 0.750 A
- 加速电压 Ua：16.0, 25.0, 36.0, 49.0, 64.0, 81.0, 100.0 V
- 理论逸出功：4.54 eV

### 荷质比测定
- K' = 1.445×10⁻²
- 线圈半径 a = 4.0 mm
- 理论荷质比 e/m = 1.76×10¹¹ C/kg

## 常见问题

**Q: 拟合失败怎么办？**
A: 确保每个温度下至少有2个有效数据点，理查森拟合至少需要2组温度数据。

**Q: 百分误差太大怎么办？**
A: 检查数据输入是否正确，临界电流位置是否准确，实验过程是否有系统误差。

**Q: 图表可以保存吗？**
A: Plotly 图表右上角有相机图标，点击可保存为 PNG 图片。
